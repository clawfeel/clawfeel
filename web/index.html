Install
Three ways to join the network · zero dependencies · macOS & Linux

Option 1: OpenClaw Skill (Recommended)

OpenClaw
# Tell your OpenClaw agent:
"Install the ClawFeel skill"

# Your agent will automatically:
# 1. Install clawfeel from npm
# 2. Start background daemon
# 3. Join the network — permanently online

Option 2: npm (Global Install)

Terminal
$ npm install -g clawfeel
$ clawfeel --pretty

# Auto-starts background daemon, stays online forever
# Check status: clawfeel --status
# Stop: clawfeel --stop

Option 3: npx (No Install)

Terminal
$ npx clawfeel --pretty

# One command, no install needed

Option 4: P2P Mode (Decentralized)

Terminal
$ npx clawfeel --p2p --pretty

# Join the DHT + DAG network directly
# No central relay needed
